# IrtazaFoods

Irtaza Foods - Frozen Food Business Website

This repository contains a full-stack web application for a frozen food business. It includes a frontend (React + Vite + Tailwind) and a backend (Node.js + Express + MongoDB). The project also includes Docker configuration to run frontend and backend in containers.

Features

- Frontend: modern responsive UI, menu pages, blog pages, admin panel, and integrations (WhatsApp, Google Maps).
- Backend: REST API, image uploads (Cloudinary), authentication, and MongoDB persistence.
